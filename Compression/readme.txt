This program compresses and decompresses files according to the specs
of the project1 instructions.

1) Compile the proj1.java file (javac proj1)
2) Include input and output files as command line args.

First arg: file to be compressed/uncompressed.
Second arg: output file destination.
	
Program will check if input file exists and prompt for new path if not.
Program will check if output file exists and prompt if OK to overwrite.

Example: java proj1 input1.txt input1_comp.txt
	 java proj1 input1_comp.txt input1_decomp.txt
